EEM package

EEM is a package for R. EEM reads raw EEM data and prepares it for further analysis. The package is available on CRAN and github.

cheng package

cheng is a package for R. It contains many misc functions, mostly including those used to support further EEM analysis. The functions are created for self-use and occasionally moved to `EEM` package once they are deemed to be important. The package is available on github.
